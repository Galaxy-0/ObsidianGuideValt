## 🔧 配置建议

### Obsidian Copilot设置
- **Embedding模型**：推荐硅基流动的`Qwen/Qwen3-Max-Embedding-4B`（免费）
- **Chat模型**：推荐DeepSeek R1或Google Gemini Flash
- **索引策略**：设置为"ON MODE SWITCH"
- **排除规则**：排除临时文件和缓存目录

### 检索优化
- **提问方式**：使用具体、明确的问题
- **关键词**：包含文档中存在的术语
- **分解复杂**：将复杂问题分解为多个简单问题
